== Description ==

Get visual and editorial content in a few clicks with our Artificial Intelligence Author. All content is unique and original. Simply modify and publish. No more blank pages!

Generate great content for your product or blog in a few clicks.

### Features

* Create blogs
* Create product descriptions
* Generate images relevant to your content

InsapptionAI makes use of [OpenAI's GPT-3.5 model and DALL-E](https://openai.com/blog/openai-api/) through the API services provided by [ai.insapption.com](https://ai.insapption.com/).

### External Service Usage

This plugin uses the API services of [InsapptionAI](https://ai.insapption.com/) for the following purposes:

- Linking user-generated API keys to their account
- Unlinking API keys
- Showing content creation options
- Creating content within WordPress

For more information, please refer to the [Terms of Service](https://ai.insapption.com/terms-of-service) and [Privacy Policy](https://ai.insapption.com/privacy-policy) pages of [InsapptionAI](https://ai.insapption.com/).

== Installation ==

### Download the Insapption AI ZIP file.

1. Log in to your WordPress admin panel.   
2. Go to WordPress Plugins page and click on Add New button, then Upload the .zip file that you downloaded.
3. Then click "Install Now" and then "Activate"
4. Create a new page/post and insert the Insapption AI block to start generating AI content

That's it! Insapption AI will now be enabled on your site!
